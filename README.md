# Voytana 2.0 — Affiliate Ready

Versione adattata per il repository attuale di Voytana.

## Struttura semplificata
Tutti i file principali sono nella root:
- index.html
- stili.css
- app.js
- favicon.svg
- sito.webmanifest
- pagine informative e legali
- robots.txt
- sitemap.xml
- vercel.json

Non serve creare la cartella `assets`.

## Installazione
1. Elimina dal repository i vecchi file duplicati:
   - styles.css
   - script.js
   - eventuali vecchi index.html
2. Carica tutti i file di questa cartella nella root del repository.
3. Conferma il commit.
4. Attendi il deploy automatico di Vercel.

## Importante
- Conserva `hero-voytana.mp4` solo se vuoi riutilizzarlo in futuro.
- Le pagine Privacy e Cookie sono modelli da personalizzare.
- Non dichiarare partnership finché non sono approvate.
- Quando arrivano i link affiliati, aggiorna la funzione `openBooking()` in `app.js`.
